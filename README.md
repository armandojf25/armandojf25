### Lullovi - Baby Tracker & Log

> Complete baby tracking ecosystem with a mobile app in **16 languages** across **102 countries**, Deep Learning-powered cry detection, first-year educational guide, and admin dashboard.

<div align="center">

[![Website](https://img.shields.io/badge/Website-lullovi.com-8B6F47?style=for-the-badge&logo=safari&logoColor=white)](https://lullovi.com)
[![App Store](https://img.shields.io/badge/App_Store-Download-0D96F6?style=for-the-badge&logo=apple&logoColor=white)](https://apps.apple.com/us/app/lullovi-baby-tracker-log/id6760706971)
[![Google Play](https://img.shields.io/badge/Google_Play-Download-414141?style=for-the-badge&logo=googleplay&logoColor=white)](https://play.google.com/store/apps/details?id=com.lullov)

</div>

A full-stack platform comprising:
- **Mobile app** - Flutter (iOS, Android, Apple Watch) with voice logging, offline support, and real-time family sync
- **Backend** - Rust/Axum API with PostgreSQL, pgvector semantic cache, HMAC-SHA256 auth, AES-256-GCM PII encryption
- **Deep Learning** - cry detection service + nap prediction
- **Infrastructure** - Docker Compose, Traefik reverse proxy, NATS JetStream messaging, Nginx, Cloudflare, Ubuntu Server on IONOS VPS
- **Security** - Rate limiting, HSTS, CSP headers, Play Integrity, App Attest
- **Services** - RevenueCat subscriptions, Shorebird OTA updates, FCM push notifications, Mixpanel analytics, Sender.net transactional email
- **Web** - Promotional landing page, React 19 admin dashboard, educational guide site
- **Testing** - k6 load testing, AWS SageMaker

### Codazzy - Infrastructure Monitoring

> Real-time infrastructure monitoring system with ML-powered predictions and AI-generated reports.

Full-stack monitoring built with **Rust/Axum** backend and **Next.js 15** dashboard. Features real-time metric collection via lightweight agents, time-series storage with InfluxDB, and ML anomaly prediction using Amazon Chronos 2. Includes automatic network topology discovery via SNMP, WebSocket streaming, and OpenAI integration for intelligent report generation. Supports GPU acceleration.
